## Usage

### Fetch the package

### View package content
`kpt pkg tree sample`
Details: https://kpt.dev/reference/cli/pkg/tree/

### Apply the package
```
kpt live init sample
kpt live apply sample --reconcile-timeout=2m --output=table
```
Details: https://kpt.dev/reference/cli/live/
